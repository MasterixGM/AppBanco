README:

Nombre de la Aplicación: AppBanco

Descripción:
La AppBanco es una aplicación para usuarios de un banco. Proporciona funciones básicas de registro e inicio de sesión, y presenta una interfaz gráficapara mejorar la experiencia del usuario.

Características Principales:

Registro de nuevos usuarios con validación de campos.
Inicio de sesión con verificación de credenciales.
Interfaz de usuario amigable con saludo personalizado y fecha de ingreso.
Redirección a una ventana principal después del inicio de sesión exitoso.
Requisitos del Sistema:

Java 8 o superior instalado.
Entorno de ejecución JavaFX.

Instrucciones de Uso:

Ejecute la aplicación ejecutando la clase AppBanco.
Complete los campos y pulsa el Boton de Registro para Registrarte o Pulsa LOGIN para iniciar sesión.
Después del inicio de sesión, se abrirá una ventana principal con información personalizada (TODO).
